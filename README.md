# Django Blog Project

This is a Blog Web Application built using Django. Users can view blog posts and manage content through the Django admin panel. 
The project demonstrates Django concepts such as models, views, templates, and database integration.

## Features

- User registration and management
- Edit and manage users
- Admin panel for managing content
- Create and manage blog posts
- Responsive user interface using Bootstrap
- Database integration using Django ORM

## Technologies Used

- Python
- Django
- HTML
- CSS
- Bootstrap
- MySql



## Future Improvements

- User login and registration
- Comment system
- Blog categories
- Search functionality
